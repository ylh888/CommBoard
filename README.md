CommBoard v1.0
=========

Communication Board for Ability

THIS IS THE AUTOSCAN VERSION

it will also activate by direct clicking

While in autoscanning, use 

- 's' to select item
- 'p' to pause/unpause
- 'v' for sound on/off
- 'd' to show debug messages
